# 🚀 AI Sales Analyzer

🔗 Live App: https://ai-sales.streamlit.app/

## 📌 Overview
AI Sales Analyzer is an interactive web application that allows users to upload CSV/Excel datasets and instantly generate insights, visualizations, and predictions.

## ⚙️ Features
- 📊 Automatic dataset analysis
- 📈 Interactive charts using Plotly
- 🤖 AI-style chat for data insights
- 🔮 Sales prediction using Machine Learning
- ☁️ Deployed using Streamlit Cloud

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
